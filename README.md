# ChaosDash

A productivity dashboard designed for ADHD workflows.

ChaosDash combines quick capture, task prioritization, focus timers, routines, planning tools, and AI-assisted helpers into a single interface built for the way chaotic brains actually work.

## Key Features

- Quick capture for thoughts, tasks, and voice input
- Do Now + Inbox workflow for limiting active tasks
- Focus mode with built-in timer and session notes
- AI-powered helpers for breaking down tasks, estimating time, softening tone, and rewriting messages
- Routines, SMART goals, mood tracking, and day planning
- Research and news aggregation for ADHD-related resources
- Sticky notes, brain dump tools, and local-first data handling

## Tech Stack

- React + TypeScript
- Tailwind CSS + shadcn/ui
- Node.js + Express
- PostgreSQL
- AI-assisted workflow tools
